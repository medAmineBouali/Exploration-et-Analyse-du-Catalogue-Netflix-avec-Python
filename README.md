🎬 Netflix Data Analytics : Stratégie \& Tendances

Ce projet propose une analyse exploratoire (EDA) approfondie du catalogue Netflix à l'aide de Python, visant à décrypter la stratégie de contenu de la plateforme.



🛠️ Stack Technique

Analyse : Pandas, NumPy



Visualisation : Seaborn, Matplotlib



📊 Insights Clés \& Résultats

L'analyse a permis d'identifier les piliers stratégiques de Netflix :



Pivot Stratégique (2016-2019) : Explosion des ajouts avec une priorité donnée aux Séries pour maximiser la fidélisation des abonnés.



Contenu Adulte : La majorité du catalogue est classée TV-MA, ciblant principalement un public adulte, avec un potentiel de croissance sur le segment jeunesse.



Géographie : Domination des États-Unis, suivis de l'Inde et du Royaume-Uni. Le contenu international (Dramas, International Movies) est au cœur de la diversification.



Genres : Prédominance des contenus émotionnels (Dramas) et des productions internationales.



Talents : Identification de figures récurrentes dans le cinéma international (ex: Anupam Kher, Shah Rukh Khan), soulignant l'importance de certains marchés pivots.



🚀 Conclusion Métier

Netflix a délaissé son modèle de simple distributeur pour devenir un producteur agressif (Netflix Originals), misant sur les séries et l'expansion internationale pour dominer le marché du streaming.
